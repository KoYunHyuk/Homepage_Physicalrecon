Platform Development

CHETAM

Platform Service

CHETAM은 PTI(Physical Threat Intelligence)를 기반으로 물리적 공간의 위협 정보를 수집·분석하고, 사용자에게 실시간 안전 정보를 제공합니다

서비스 소개

CHETAM 모바일 App에서 사용자가 직접 위협 정보를 제보하고, 제보데이터 바탕으로 위치기반 SOS 등의 부가서비스를 제공합니다.

시연 영상 바로보기
Main Service

위협정보 제보

사용자가 모바일 App에서 주변 위협 상황을 직접 제보하는 기능입니다. 위치 기반으로 위협 정보를 등록하고, 텍스트와 이미지 등 현장 정보를 함께 전달할 수 있습니다.

Membership Service

SOS 위치 공유

친구로 등록된 멤버 사이에서 SOS 요청을 보내면, 요청자의 위치가 보호그룹에 표출되는 기능입니다. 위험 상황에서 빠르게 위치를 공유하고 도움을 요청할 수 있습니다.

플랫폼 구성

모바일 App 환경에서 위협 정보를 수집하고 분석한 뒤, 사용자에게 경보와 행동 정보를 제공하는 구조로 구성됩니다.

Module 01

위협정보 수집

OSINT, HUMINT, SOCMINT와 사용자 위치 기반 제보를 결합해 주변 위협 데이터를 수집합니다

Module 02

다층 AI 분석

범죄, 사고, 재해 이력과 실시간 데이터를 계층별로 분석해 위협 위치와 심각도를 도출합니다

Module 03

모바일 경보 표출

Map 기반 위협 경보, 알림 팝업, 보호자 위치 공유, 주변 사용자 도움요청 기능을 제공합니다
